Example 1

Flexbox

Flexbox is the easiest way to center an element both vertically and horizontally. When you do not know the width and height of the child or the parent div, you can use display: flex; on the parent container and align-items: center; centers the child vertically and justify-content: center; centers it horizontally.

.container { 
          display: flex;
          align-items: center;
          justify-contents: center;
      }
center me!
Example 2

Grid

It is possible to make the content centered using grid by giving its parent display grid property. Grid has a shortcut place-items: center; which makes the child both horizontally and vertically centered.

NOTE: You can also use margin: auto; on the child element to center it inside a grid container 

.container { 
          display: grid;
          place-items: center;
      }
center me!
Example 3

Position Absolute

Give the parent container position: relative; and then make the child position: absolute;. Now you can position it in center using left and top rule. However defining top: 50%; and left: 50%; moves the top left corner of the child at 50% which is not exactly center. So to make it perfectly centered, we need to also specify a transform: translate(-50%, -50%); which moves the child towards left 50% of its own width, and top 50% of its height. Which gives us perfectly centered content.

#example3 {
        position:relative;
      }
      
      #example3 div {
        position: absolute;
        top:50%;
        left:50%;
        transform: translate(-50%, -50%);
      }
center me!
Example 4

Using table

This technique can be used if we do not worry about child div stretching vertically and horizontally in the parent div. We can simply make the parent div behave as table by display: table; and the child div a cell inside table by specifying display: table; on child element. Now we can use the property vertical-align: middle; to align the content vertically centered. If you want to align it horizontally centered too, you can use text-align: center; in the child.

#example4 {
        display: table;
      }
      #example4 .center-me {
        display: table-cell;
        vertical-align: middle;
        text-align: center;
center me!
Example 5

Auto Margin

This method works on the content itself and is not dependent on container div. But it only works when the width of the content is known or defined in the CSS. Defining left and right margin as auto tells css to always have the same margin on left and right, which makes it align centered. margin: auto; only works for centering horizontally, it does not work vertically.

 #child4 { 
          width: 100px;
          margin: 0 auto;
      }
center me!
Example 6

Text Align

The text-align property in CSS is used for aligning the inner content of a block element. When you add text-align: center; to the parent element, its inside content centers between the left and right edges. White space on the left and right sides of each line should be equal. While text-align certainly affects text, it affects all inline or inline-block elements in that container. 

 #example6 { 
          text-align: center;
      }
center me!
Example 7

Padding with calc

If you know the height of your content, you can also use padding in combination with calc() function in CSS. When we assign 50% padding on both sides it leaves no room for the content. Then we use Calc() function to subtract the width of our content. For example: if we have a content of 100px, we subtract 50px from both left and right padding.
Its not a very effective way, but just to mention that it can be useful to center content without parent container. It can be used in very creative way and as a fallback for grid. 

 #example7 .center-me { 
          height: 100px;
          padding: 0 calc(50% - 20px) ;
      }
center me!
